# Wolf-Gang Engine
A magical 2d game engine for creating extremely interactive rpg-styled games.

### Features
- Animations
- Entirely scripted scenes using AngelScript
- Simple save system
- Tilemap
- Sound
- Narrative
- Builtin Tilemap and collisionbox editor
- Fast :D
